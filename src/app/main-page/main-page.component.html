Search Open Data in One Box

What is Singular Data Net?

Singular Data Net is an experimental project to provide one single search box for datasets from multiple data providers. With a data engine that constantly monitor the update of data providers, we are able to build a unified search index with standarized open data metadata. The user is able to search data from all supported data providers at one click and see results in one format.

Currently, the project supports a variety of open data platforms:

  • ArcGIS Open Data
  • CKAN
  • DKAN
  • GeoNode
  • Junar
  • Socrata

A full list of supported open data portals (100+) could be found here .

Why is it developed?

Open data is not open if it cannot be discovered by the user.

Every data analyst has experienced the pain of finding the correct open data. Though open data is becoming a popular concept and more action is being taken, the discovery of data doesn't seem as easy as it looks like. Of many factors that contributes to this problems, there are two significant factors:

  • Open data is a highly popular field that attracts many governments and organizations to participate. The increasing number of data providers also increases the difficulty for a user to identify the source of a particular dataset.
  • Since it is still an reapidly developing area, many different standards and practices co-exist. The various ways to search and present data result in confusion and therefore makes the data discovery again uneasy.
