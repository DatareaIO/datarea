What?

Datarea.io is a search engine that allows you to search data across open data portals at multiple networks.

Why?

T he number of open data portals and open data has been dramatically increasing in the recent years. With the advancing regulation and technology, more individuals and organizations are joining the force of data opening.

We are excited, but yet concerned, to see this future.

A recent survey shows that there are more than 2,600 data portals around the world. Most data portals are either disconnected to each other, or only within a local network run by a portal service provider.

It needs time, experience, and even connections for the data user to identify the correct data portal and find the correct data.

It should be simpler.

How?

Datarea.io provides the one-click-and-search-all service.

By collecting open data metadata from more than 700 portals in different open data networks, we are able to build one open data network that includes the majority of the world of open data and just search in it.

You only need to type your keywords in one text box and Datarea.io will present you the correct dataset from the correct portal.
