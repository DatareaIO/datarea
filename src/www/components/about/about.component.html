Hi, I am Haoliang Yu, the developer behind Datarea.io.

Suitable data is uneasy to find. According to a recent report from OpenDataSoft, more than 2,600 open data portals are in the world and the number keeps increasing. There is on doubt that the increasing data opening creates more potential for data analysis and research, but the rapidly increasing open data and the more distributed open data network has created questions for us:

  • Where can I find the wanted data?
  • How do I know whether it's appropriate for my project?
  • How many possible dataset can I find?

It takes time, experience, expertise, even social connections to accomplish the data finding task. All these together makes up a larget part of professional work for a data analyst or scientist.

This work shouldn't be this hard and this is why I create the application Datarea.io, the anywhere door to what you want. No more searching data for hours. No more asking friends of friends for data information. It should be an entrance for the extensive open data network and a tool that helps you reach 1,000+ open data portals with just a few clicks.
